O CÓDIGO POR TRÁS DA SAÚDE

Role para baixo ou use as setas para avançar

o que é

Vimos o ambiente de uma empresa de tecnologia de ponta.
Um espaço de colaboração, foco e muita inovação.

maria da penha

1. ANTES (A Ideia)

Tudo começa com uma necessidade. Há conversas, geração de demandas e documentos para definir o que será criado.

2. DURANTE (A Construção)

A equipe começa a projetar, programar e, crucialmente, testar cada parte do sistema para garantir a qualidade.

3. IMPLANTAÇÃO (O Lançamento)

O sistema é instalado e distribuído para os usuários. É a hora da verdade, quando o código vai para o mundo real!

4. DEPOIS (A Evolução)

O trabalho nunca para! É preciso prestar suporte, monitorar, fazer melhorias e gerenciar o sistema continuamente.

objetivos

  • Ver que o que aprendemos em aula ACONTECE DE VERDADE nas empresas.
  • Entender como funcionam as Squads Multidisciplinares, com programadores, designers e gestores.
  • Descobrir como a tecnologia resolve problemas reais, suportando uma demanda de trabalho gigante.

tipos

  • Firebase & Flutter: A dupla do Google para criar APPs para celular.
  • Angular & .NET: Combinação poderosa para o front-end e o back-end dos sistemas.
  • PostgreSQL: O "cérebro" que guarda todos os dados de forma organizada.
  • Amazon AWS & Google Cloud: A "casa" dos sistemas, rodando na nuvem 24/7.
  • Docker: As "caixinhas" que garantem que o código funcione em qualquer lugar.
